# Battle pirates Auto script

The python script is meant to automate specific elements of the game. Saving You time by not needing to do repetitive and boring tasks. This includes: crew rolling, target hunting, at base fleet repairing.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install the required libraries. If You are not on a windows platform, you can skip winotify and comment out the last except block of the script in .\BP_fleet_manager.py

```bash
pip install requests websocket-client winotify
```

## Usage

You can write whatever scenario you have in mind, using the functions found inside the script.

```python
# starts the script
python .\BP_fleet_manager.py

```
